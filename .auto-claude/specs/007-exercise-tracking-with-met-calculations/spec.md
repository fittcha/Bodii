# Exercise Tracking with MET Calculations

Exercise logging with activity type selection, duration input, and automatic calorie burn calculation using MET (Metabolic Equivalent of Task) values. Track both cardio and strength training with appropriate calculations.

## Rationale
Completes the calorie in/out equation by tracking expenditure. Integrated with body composition and diet data to show net daily balance - something fragmented apps require multiple tools to achieve. Addresses user goal of 'maintaining consistent daily records of health metrics'.

## User Stories
- As a fitness enthusiast, I want to log my workouts so that I can track my activity level
- As a user, I want automatic calorie burn calculations so that I know my daily expenditure

## Acceptance Criteria
- [ ] Users can select from common exercise types (running, cycling, swimming, weights, etc.)
- [ ] Duration input in minutes with automatic calorie calculation
- [ ] MET values used for accurate calorie burn based on body weight
- [ ] Exercise records linked to daily log for aggregation
- [ ] Daily exercise summary shows total activity calories
- [ ] Custom exercise entry available for unlisted activities
